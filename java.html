Full Stack Development Syllabus

Month 1: Foundations (Day 1 - Day 20) ▼
  • Day 1-3: Introduction to HTML5, Basic HTML structure, Document structure, DOCTYPE declaration
  • Day 4-6: HTML Elements: Paragraphs, Headings, Lists (ordered/unordered), Links, Images
  • Day 7-9: HTML Forms: Input types, buttons, text areas, checkboxes, and radio buttons
  • Day 10-12: Semantic Elements: <header>, <nav>, <section>, <article>, <footer>
  • Day 13-16: Tables and Lists: <table>, <thead>, <tbody>, <tfoot>
  • Day 17-20: Multimedia: Embedding audio, video, and iframes
Month 2: Advanced UI/UX Design (Day 21 - Day 40) ▼
  • Day 21-24:
  • Day 25-28: Box Model: Margins, Padding, Borders, and Content
  • Day 29-32: CSS Layout: Display, Positioning, Flexbox
  • Day 33-36: CSS Grid: Introduction to CSS Grid layout
  • Day 37-40: Responsive Design: Media queries and breakpoints, CSS animations
Month 3: JavaScript (Day 41 - Day 60) ▼
  • Day 41-44: Introduction to JavaScript: Basics, variables, data types
  • Day 45-48: Conditional Statements, Loops
  • Day 49-52: Functions, JavaScript Events
  • Day 53-56: DOM Manipulation, Form Validation
  • Day 57-60: Advanced JavaScript, ES6 features
Month 4: React.js (Day 61 - Day 80) ▼
  • Day 61-64: Introduction to React.js, React components, JSX
  • Day 65-68: State Management, useState, useEffect hooks
  • Day 69-72: Handling Events in React
  • Day 73-76: React Router, SPA navigation
  • Day 77-80: Fetching Data from APIs, Building a small React project
Month 5: Python Backend Development (Day 81 - Day 100) ▼
  • Day 81-84: Introduction to Python, setup, data types
  • Day 85-88: Python Control Flow: if-else, loops
  • Day 89-92: Flask setup, routing, templates
  • Day 93-96: Flask and Forms, Database connectivity
  • Day 97-100: Introduction to Django, Models, Views, Templates
Month 6: Full Stack Integration and Deployment (Day 101 - Day 120) ▼
  • Day 101-104: Frontend and Backend Integration
  • Day 105-108: Authentication: JWT, OAuth
  • Day 109-112: Backend API Development, RESTful APIs
  • Day 113-116: Deployment Basics: Hosting on Cloud
  • Day 117-120: Final Project: Building a full-stack web application
